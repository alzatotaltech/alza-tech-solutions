ALZA Website - GitHub Pages Upload Instructions

1. Unzip this file.
2. Open the unzipped folder.
3. Upload ALL CONTENTS directly to the root of your GitHub repository.

Your repository root must show:
- index.html
- services.html
- ai-solutions.html
- industries.html
- projects.html
- about.html
- contact.html
- privacy.html
- terms.html
- assets/
- .nojekyll

Do not upload the zip file itself.
Do not upload the parent folder only.

GitHub Pages Settings:
Source: Deploy from branch
Branch: main
Folder: /root

After saving, wait 1-2 minutes and refresh.
