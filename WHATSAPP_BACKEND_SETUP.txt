IMPORTANT: Live delivery to your WhatsApp Messenger requires a backend service.

This website now opens an on-site chat window only. It does not redirect visitors to WhatsApp.

To send enquiries live to your WhatsApp Messenger, connect:
1. WhatsApp Business Cloud API, Twilio WhatsApp, WATI, Interakt, or another WhatsApp API provider.
2. Create a backend endpoint, for example: /api/send-whatsapp-enquiry
3. In assets/js/main.js, set:
   const WHATSAPP_BACKEND_ENDPOINT = "/api/send-whatsapp-enquiry";

The static frontend cannot securely send WhatsApp messages directly because WhatsApp API tokens must not be exposed in browser JavaScript.
